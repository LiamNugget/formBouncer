# Processed Data Schema

## Format: JSON Lines (JSONL)

One JSON object per line. No trailing commas. UTF-8 encoded.

Files:
- `train.jsonl` — 80% of data, used for model training
- `test.jsonl`  — 20% of data, used for evaluation only (never seen during training)

Both files are generated by running `ml/pipeline/preprocess.py` and should not be edited by hand.

---

## Fields

| Field          | Type   | Values                                | Description                                              |
|----------------|--------|---------------------------------------|----------------------------------------------------------|
| `text`         | string | any                                   | Cleaned plain text of the message body. Headers stripped, whitespace normalised, max 10,000 chars. |
| `label`        | string | `"spam"` or `"ham"`                  | Ground truth classification.                             |
| `source`       | string | `"spamassassin"` `"enron"` `"ling_spam"` | Which corpus this record came from.                 |
| `processed_at` | string | ISO 8601 datetime                     | When this record was written by preprocess.py.           |

---

## Example Records

```jsonl
{"text": "Congratulations! You have won a $1,000,000 prize. Click here to claim.", "label": "spam", "source": "spamassassin", "processed_at": "2026-02-22T09:00:00"}
{"text": "Hi John, just following up on the meeting notes from yesterday.", "label": "ham", "source": "enron", "processed_at": "2026-02-22T09:00:00"}
{"text": "Buy cheap medications online. No prescription needed. Discreet shipping.", "label": "spam", "source": "ling_spam", "processed_at": "2026-02-22T09:00:00"}
```

---

## Adding New Data Sources

1. Create a new source module in `ml/pipeline/sources/your_source.py`
2. Implement a `load(unprocessed_dir: Path) -> list[dict]` function
3. Each dict must contain at minimum: `text`, `label`, `source`
4. Register the source in `ml/pipeline/preprocess.py`

## Regenerating

```bash
python ml/pipeline/preprocess.py
```

This will re-download missing corpus files and overwrite train.jsonl and test.jsonl.
